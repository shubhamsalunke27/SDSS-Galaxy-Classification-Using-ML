Need Help With Input Values?

All features should be numerical values from actual SDSS measurements. Here's a guide for each input:

specobjid
Spectroscopic Object ID - unique identifier for the galaxy
Example: 1000000
u
Magnitude in ultraviolet u-band (brightness measurement)
Example: 18.5
modelFlux_i
Model flux in i-band (near-infrared light intensity)
Example: 2500
modelFlux_z
Model flux in z-band (infrared light intensity)
Example: 1800
petroRad_u
Petrosian radius in u-band (physical size in u-band)
Example: 3.2
petroRad_g
Petrosian radius in g-band (physical size in g-band)
Example: 3.5
petroRad_i
Petrosian radius in i-band (physical size in i-band)
Example: 3.8
petroRad_r
Petrosian radius in r-band (physical size in r-band)
Example: 3.6
petroRad_z
Petrosian radius in z-band (physical size in z-band)
Example: 3.4
redshift
Redshift value (indicates distance and age of galaxy)
Example: 0.05
